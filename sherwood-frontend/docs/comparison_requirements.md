# Results Similarity Display Requirements Summary

The results detail page should have a means to compare similar results side by side with that of the parent result.

## Results Detail Page changes

The similar results table on the results detail page should have a clickable element on each row, allow the user to compare the the similar result they've clicked on against the main result for the page.

- Add a new column at the end with a clickable element to compare.
- The new column does not need a title, but should show an icon. This icon is what the user clicks on. Pick an icon that conveys comparing.
- Click the icon should show a dismissable popup, showing the similar result and the main result side by side with some fields. 
- Main result should be on the left, similiar result on the right.

The following fields should be displayed :
- rule id
- location
- line number
- description
- snippet

Under the 2 result display panes, show the similarity and reason fields from the similar result.
